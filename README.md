# ELI12 — Explain Like I'm 12

Offline web app that explains topics in simple language and can search inside your uploaded theory books (.txt).

## Features
- Ask a question or paste textbook text
- Simple explanation (levels 1–3)
- Formulas / key facts for common topics
- Local Knowledge Base: upload .txt books → get citations/snippets (works in-browser)
- Useful learning links

## How to use
1. Open the website (GitHub Pages link).
2. Type your topic/question and/or paste text.
3. (Optional) Upload your books as .txt via “Upload .txt”.
4. Press **Explain**.

## Tech
HTML + CSS + Vanilla JS (no backend).
